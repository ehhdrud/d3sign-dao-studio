## 🚩 FUNCTIONS

-   여려 AI 아티스트와 예술 작품 소개
-   Media Query를 이용한 반응형 웹사이트
-   사용자의 메타마스크 계정을 조회하여 현재 연결된 네트워크의 주소와 자산 확인 가능

> #### ⚠ 주의사항
> 인터넷 지갑에 접근하기 위해 메타마스크 로그인이 필요합니다.<br/>
> 메타마스크가 설치되지 않은 환경, 모바일 환경에서는 메타마스크를 사용할 수 없기 때문에 해당 기능이 존재하지 않습니다.
